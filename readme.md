# Kubescape and Kube-bench 🔊

Kubescape is a tool used for scanning for cluster hardening and benchmark compliance with NSA and the MITRE ATT&CK framework post deployment. Kube-bench is another tool used for compliance with CIS benchmarks checking for cluster level security configuration 

Kube-bench -> cluster hardening 🛡️
kubescape -> 📊 Compliance
