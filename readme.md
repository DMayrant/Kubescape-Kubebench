# Kubescape and Kube-bench 🔊

Kubescape is used to scan Kubernetes clusters for misconfigurations and security risks, aligning with frameworks such as NSA hardening guidelines and MITRE ATT&CK.

Kube-bench is used to validate Kubernetes nodes and control plane components against CIS (Center for Internet Security) benchmarks, ensuring proper cluster hardening.

Kubescape → 📊 Cluster misconfigurations & compliance (NSA, MITRE)

Kube-bench → 🛡️ Node & control-plane hardening (CIS benchmarks)
